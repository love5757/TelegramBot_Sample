# TelegramBot_Sample with spring + mysql
커맨트 패턴 학습용으로 만들어진 봇 임 그외 다양한 학습의 흔적들이 존재하여
참고용으로만 봐주세요.

커맨드 이외의 메세지가 들어 오면 심심이가 대답을 합니다.



#Command List
- start - 시작(봇 활성화)
- stop - 종료(봇 비활성화)
- hello - 인사
- lotto - 로또 번호 조회 및 추천 번호 보여주기(11번가 네트워크 불가로 안됨)
